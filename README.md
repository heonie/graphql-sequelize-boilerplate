# GraphQL Server boilerplate

ApolloServer + Express + Sequelize

## Setup
```
yarn install
```

## Development
```
yarn dev
```

## Build and Start
```
yarn build
yarn start
```

## Environment Variables
For running on the `production` mode, you have to these environment variables:

* NODE_ENV=production
* DB_HOST
* DB_NAME
* DB_USERNAME
* DB_PASSWORD
* PORT (Optional, default: 4000)
